Operating Systems

Summary Linux Windows Mac OS


Windows OS is developed by Microsoft Corporation. Windows was the first

OS to feature a graphical user interface(GUI) that could run on IBM PCs. Windows is the most popular

out of the three major Operating Systems with about 90% of personal computers

running some version of it. The first version of Windows was released in

1985. There have been many versions of Windows developed since then including

Windows 7(considered by many to be the zenith of the OS). The most recent version

of Windows is Windows 10.
