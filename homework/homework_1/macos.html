Operating Systems

Summary Linux Windows Mac OS


Mac OS is an operating system developed by Apple Inc. Mac OS was first released in 1984. It was one of the first OSs to have a graphical user interface(GUI). Many aspects of Mac OS were licensed out to Microsoft for the creation of Windows OS. A major shift in design for Mac OS came about in 2001 with the release of Mac OS X. Mac OS High Sierra is the lastest version of Mac OS to be released.
