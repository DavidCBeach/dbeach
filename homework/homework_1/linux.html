Operating Systems

Summary Linux Windows Mac OS


Linux is the least popular out of the three main Operating Systems but Linux is also the only one that's free. Linux is open source. The lack of an accompanying cost makes Linux a great option large scale server setups. Unlike the other OSs Linux has a large variety current of distributions that the user can choose from. Some of the most popular distributions of Linux are:
  • Ubuntu Linux
  • Linux Mint
  • Arch Linux
  • Deepin
  • Fedora
  • Debian
  • openSUSE
Every distribution has its own verson of the traditional desktop environment or omit it all together(Arch Linux). Installing software on Linux can be a unique experience as well. Many distrobutions are designed to mainly install software using terminal commands.
